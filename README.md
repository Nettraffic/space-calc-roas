# Space Edu - Calculadora ROAS Inteligente 🚀

Sistema profissional para cálculo e análise de ROAS (Return on Advertising Spend) em campanhas de marketing digital.

## 🎯 Funcionalidades

### ✨ **Core Features**
- **Cálculo automático de ROAS** para campanhas
- **Dashboard interativo** com KPIs principais
- **Gráficos de performance** (Barras, Pizza, Linha)
- **Filtros por data** para análise temporal
- **Gestão completa de campanhas** (CRUD)

### 🎨 **Interface**
- **Design moderno** com identidade visual preto e vermelho
- **Modo escuro/claro** para melhor experiência
- **Multi-idioma** (Português/Inglês)
- **Interface responsiva** para todos os dispositivos

### 📊 **Analytics**
- **KPIs em tempo real**: ROAS Total, Receita, Investimento, Lucro
- **Gráficos interativos** com Recharts
- **Análise por fonte de tráfego**
- **Tendências temporais** das campanhas

### 🛠️ **Funcionalidades Técnicas**
- **Exportação de dados** (CSV)
- **Sistema de filtros** avançados
- **Validação de formulários**
- **Dados demo** para demonstração

## 🚀 **Como usar**

## 📊 **Métricas Calculadas**

### **ROAS (Return on Advertising Spend)**
```
ROAS = Receita ÷ Investimento
```

### **Interpretação**
- **ROAS ≥ 4.0**: 🟢 Excelente performance
- **ROAS 2.0-3.9**: 🟡 Performance adequada  
- **ROAS < 2.0**: 🔴 Atenção necessária

### **Outras Métricas**
- **Lucro**: Receita - Investimento
- **ROI**: ((Receita - Investimento) ÷ Investimento) × 100
